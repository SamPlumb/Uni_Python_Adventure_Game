# KeyError: The Keeper of the Golden Key

In this text-based adventure game, your goal is to explore rooms and collect items. Some items could be useless, help
you progress or enable your escape!

## How to play

All player inputs are made by typing a number into the terminal. Each menu will display a number before each option, for
example, "1. Start Game"; the player should type that number to select the option in this instance "1".

## Requirements

KeyError requires Python 3.14 available at [python.org/downloads](https://www.python.org/downloads/)
